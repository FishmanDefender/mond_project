.. _data_utils:

**************
Data utilities
**************

.. currentmodule:: mond_project

Utilities for reading and utilizing the data are provided through the :mod:`data_utils` subpackage. The bulk of this subpackage is contained within the :ref:`data_read_utils <data_utils.data_read_utils>` module, which contains the function for getting the data from the the `Illustris simulations`_.

.. _Illustris simulations: http://www.illustris-project.org
